Padding & Margin

Values are in rem unit

Single value of padding & margin

Padding & Margin value will be applied on all four sides.

Padding class: pad-1,pad-2,pad-3,pad-4

Margin class: mar-1,mar-2,mar-3,mar-4

Double value of padding

The first value given in Padding will be applied to top & bottom. The second value will be applied to left & right.

Padding class:pad-1-2,pad-1-3,pad-1-4

Padding class:pad-2-1,pad-2-3,pad-2-4

Padding class:pad-3-1,pad-3-2,pad-3-4

Padding class:pad-4-1,pad-4-2,pad-4-3
